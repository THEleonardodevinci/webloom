# Webloom Prototype

A lightweight website-builder prototype inspired by Wix/Squarespace.

## Features

- Email/password sign-up + login (prototype local auth)
- Simulated Google sign-in
- Start from blank, business, or portfolio template
- Drag-and-drop blocks onto a central canvas
- Multi-page site support (add/remove/switch pages)
- Auto-save to localStorage while editing
- Preview in a new tab using `about:blank`
- Publishing/domain UI scaffold for future implementation

## Run locally

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
